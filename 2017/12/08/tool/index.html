2、截图工具

snipaste 使用文档

最常用的两个功能：

  1. 吸色 ： 该插件可以配置颜色格式，rgb或是#333两种，而第二中开发非常需要。
  2. 贴图 ： 很多时候我们需要对比一些内容，将一些内容截图贴到屏幕上就可以非常好的对比。

　贴图功能（gif截图居然绿了。。。）

FSCapture 没用过，有兴趣的可以试试。

3、思维导图

这种软件多数是付费的了，不付费的功能受限。用过下面两款。主要用来总结，归纳。

　1、免费在线ProcessOn ： 提供了很多常用的内容，还有很多其他类型的，流程图，拓扑图，不过限制文件个数，不想花钱就多注册几个账户然后在邀请吧。可以团队协作。我的邀请链接

　2、Xmind : 如果你想把思维导图导出为图片，还是用上面那个吧。

4、有道云笔记

　1、团队协作和个人笔记，文件，的存储和同步。

　2、支持 markdown 编辑，如果写文字，写心情，写文章的话，markdown还是非常nice的。

5、百度云

　备份一些比较大的破解的软件和工具。避免换电脑，换工作，装系统后又花费大量时间查找，安装。不过我还是想说一下，限速真tm恶心。(东西多的自己备个移动硬盘吧)

6、邮箱

Foxmail 邮件客户端

　不推荐qq邮箱，乱七八糟挺多的。虽然工作上一直用的是qq的。

　1、有一个个人的邮箱，各种帐号注册都用该邮箱，然后把密码设置的安全些。用手机号注册太墨迹，手机号注册那些重要的且必须的东西。越少越好，避免换号带来太多不必要的问题。

7、gifcam动态截图

　很多时候，文字和单纯的图片是不能清晰描述我们所想表达的东西，但是如果录视频又不太值当。所以用gif来解决吧。而且删除即可，不生成注册表，不用卸载

　1、gifcam教程 不知道我电脑的问题，还是怎么着，总会把边框也截进去。

　2、ScreenToGif 下载Single executable, no install ，免安装的.exe文件。点击即可使用。

8、录屏

　录屏的需求不大，但难免会有时候需要。对于不是常用的软件，其实没必要下载，所以推荐一个在线的。

　1、在线录屏： 其实也不算是在线的，毕竟还得下载，安装启动器，不过偶尔一用还是蛮不错的。

9、识别图片文字

　可以在百度检索OCR，不过大多数都是付费，对于一般简单需求，且不常用的优先考虑在线的。

12、检索工具

　everything : 相比于win自带的搜索，还是很优秀。速度快的没话说。不论是偶尔用，还是常用都非常方便，程序非常小。不拖电脑。

13、隐私工具

　Wise Folder Hider : 个人隐私很重要，同时也能更好的保护自己的文件不被乱动。，但是要记住密码。只需要使用免费的功能就行。会把你想要加密的文件隐藏。打开软件输入密码就能看到了。

14、数据回复

　DiskGenius ,对于用u盘装系统的人，应该都知道这个，很对启动盘里都带有这个，可以用来磁盘分区。也可以进行数据恢复。

　教程

　DiskGenius 密码：hcfv

简易步骤：

  1. 打开软件，选择盘符，点击左上角恢复文件。
  2. 选择仅恢复删除的文件，选择要回复的文件类型，点击开始。
  3. 下图示例，没有等待扫面完成，自己恢复时，一定要扫瞄完成后在恢复，不然可能会有遗漏。
  4. 找到被误删的文件，右键选择复制文件到。
