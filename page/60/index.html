set&map

set

  • 类似于数组，但是成员的值都是唯一的，没有重复的值。
  • Set 函数可以接受一个数组（或类似数组的对象）作为参数，用来初始化。

数组去重

set内部区分相同

NaN 任务相同，2 和 ‘2’不同，也就是说不会隐式转换。

set 方法

  • add(value)：添加某个值，返回 Set 结构本身。
  • delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
  • has(value)：返回一个布尔值，表示该值是否为Set的成员。
  • clear()：清除所有成员，没有返回值。
